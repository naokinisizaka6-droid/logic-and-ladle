Exp.14 Protein / Fish

死後硬直とATP分解：
魚の「熟成（Aging）」におけるイノシン酸生成

「新鮮な魚＝美味しい」は必ずしも真実ではない。
魚の死後、筋肉の中で静かに進行する生化学反応。
ATPが旨味成分イノシン酸に変わる時、魚は食材としてのピークを迎える。

1.0 序論：活け締めと熟成

釣り上げた直後の魚は、まだ筋肉がエネルギー（ATP）を持っており、プリプリしているが旨味は少ない。
時間が経つにつれ、ATPが分解されて旨味成分に変わる。これを意図的に管理するのが「熟成」だ。

2.0 分解モデル：ATPからIMPへ

エネルギー通貨であるATP（アデノシン三リン酸）は、死後、酵素によって分解されていく。
ATP → ADP → AMP → IMP（イノシン酸）。このIMPこそが、魚の旨味の主役である。
さらに分解が進むと、イノシン（HxR）やヒポキサンチン（Hx）になり、鮮度が落ちて臭みが出る。

[ATP Degradation Pathway]
ATP (Energy) -> ADP -> AMP
-> IMP (Umami/Inosinic Acid) **PEAK**
-> HxR (Tasteless) -> Hx (Bitter/Rotten)

3.0 制御理論：K値と熟成期間

旨味のピークは魚種やサイズ、締め方によって異なる。
神経締めをしてATPを温存した魚は、死後硬直の開始を遅らせることができ、より長く熟成させることができる。

4.0 LABORATORY PROTOCOLS：実践メソッド

Protocol A: 神経締めの効果

脳からの信号を断つことで、死後硬直までの時間を稼ぎ、ATPの浪費を防ぐ（元エネルギーを残す）。

Protocol B: 脱水処理

熟成中は水分が大敵。ピチットシートなどで余分な水分とドリップを取り除き、臭みの発生とバクテリアの増殖を抑える。

Protocol C: 魚種による違い

白身魚はイノシン酸の分解が遅く熟成に向く。赤身魚（マグロなど）は酸味が出やすいため、熟成の管理がシビアになる。
